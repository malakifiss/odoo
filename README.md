Ce projet consiste en le développement d’un module personnalisé Odoo 17 dédié à la gestion des inscriptions simples.
Il permet de centraliser les informations des inscrits et de gérer efficacement leur cycle de vie à travers une interface intuitive intégrée à l’ERP Odoo.

Le module offre les fonctionnalités suivantes :

Création, consultation et modification des inscriptions

Gestion d’un workflow simple avec trois statuts : Nouveau, Validé, Annulé

Actions métier permettant de valider ou d’annuler une inscription

Interface utilisateur claire avec vues liste et formulaire

Persistance des données via une base PostgreSQL

Le projet a été réalisé dans un cadre académique afin de mettre en pratique les concepts fondamentaux du développement de modules Odoo, notamment l’architecture MVC, la logique métier en Python, la définition des vues en XML et l’utilisation de Docker pour le déploiement de l’environnement.
